Au clic sur le bouton "Démarrer un projet" (navbar et hero), une MODAL animée 
s'ouvre avec un formulaire intelligent en 4 étapes (stepper visuel en haut).

OUVERTURE DE LA MODAL :
- Overlay sombre (rgba 0,0,0,0.7) qui fade-in
- La modal slide-up depuis le bas avec un effet spring (cubic-bezier élastique)
- Bouton X pour fermer avec rotation 90° au hover
- Fond de la modal en Charcoal (#1F2937) avec bordure subtile Peach

STEPPER EN HAUT DE LA MODAL :
- 4 cercles numérotés reliés par une barre de progression
- La barre se remplit en Peach à chaque étape validée
- Étape active : cercle plein Peach + label visible
- Étapes passées : cercle avec checkmark ✓ en Peach
- Transitions fluides entre chaque étape (slide left/right)

📐 EXIGENCES TECHNIQUES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- Fichier HTML unique avec CSS et JS intégrés dans des balises <style> et <script>
- 100% responsive : mobile-first, breakpoints 768px et 1024px
- Aucun framework externe (pas de Bootstrap, pas de jQuery) — CSS et JS vanilla purs
- Utiliser les CSS Custom Properties (variables) pour toutes les couleurs
- Utiliser CSS Grid et Flexbox pour les layouts
- Smooth scroll natif : scroll-behavior: smooth
- Performance : animations via transform et opacity uniquement (pas de layout shift)
- Accessibilité basique : aria-labels sur boutons, alt sur images, focus visible
- Les sections alternent fond Charcoal et fond Peach de façon intentionnelle 
  pour créer un rythme visuel fort et démontrer la maîtrise des inversions
- La modal est entièrement gérée en JS vanilla (open/close/steps/validation)
- Les données du formulaire sont stockées en JS (objet local) et affichées 
  dynamiquement dans le résumé final (étape 4)
- La modal doit être accessible : focus trap, fermeture avec Escape, 
  aria-modal="true"
